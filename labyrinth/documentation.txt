# команды для перемещения точки по лабиринту
1) LEFT int - сдвигает точку влево на int (целое число шагов), если int не указано: сдвиг на 1 шаг влево
2) RIGHT int - сдвигает точку вправо на int (целое число шагов), если int не указано: сдвиг на 1 шаг вправо
3) UP int - сдвигает точку вверх на int (целое число шагов), если int не указано: сдвиг на 1 шаг вверх
4) DOWN int - сдвигает точку вниз на int (целое число шагов), если int не указано: сдвиг на 1 шаг вниз
Пример:
	LEFT 5
	UP
	
# команды для печати лабиринта
5) PRINT_ON params - устанавливает режим печати лабиринта (режимы можно комбинировать)
   PRINT_ON ~params - отменяет режим печати лабиринта
   PRINT_ON ~ - отменяет все режимы печати лабиринта
         режимы печати:
         * each_step - печать на каждом шаге
         * error - печать в случае, если произошла ошибка при сдвиге точки
         * change_direction - печать при изменении направления сдвига точки
         * target - печать при достижении цели
Пример:
	PRINT_ON each_step
	PRINT_ON ~each_step error target
